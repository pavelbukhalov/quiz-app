#  QuizApp

Интерактивная платформа для проведения квизов в реальном времени.

##  Возможности

- Регистрация и авторизация (организаторы и участники)
- Создание квизов с уникальным кодом комнаты
- Добавление вопросов (текст/изображения, одиночный/множественный выбор)
- Игра в реальном времени через WebSocket
- Подсчёт баллов и лидерборд
- Личные кабинеты для организаторов и участников

## Технологии

- Backend: Node.js, Express, Socket.IO, better-sqlite3
- Frontend: HTML5, CSS3, JavaScript (vanilla)
- База данных: SQLite

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/quiz-app.git
cd quiz-app

2. Установите зависимости:
cd Backend
npm install

3. Запустите сервер:
npm run dev

4. Откройте Frontend/index.html в браузере

Роли:

Организатор — создаёт квизы, добавляет вопросы, управляет игрой
Участник — подключается по коду комнаты и отвечает на вопросы
